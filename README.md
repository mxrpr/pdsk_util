# pdsk_util

**`df` never looked this good.** pdsk_util is a fast, zero-config disk usage viewer for your terminal — color-coded usage bars, human-readable sizes (B/K/M/G/T), clean sorted output, and cross-platform support (Linux/Mac/Windows). One command, no clutter.

# installation
1. clone the repo
2. install it with pip
```
pip install .
```

# usage
```
pdsk-util              # show all volumes
pdsk-util /home        # show only /home mount point (Linux/Mac)
pdsk-util C:\          # show only C: drive (Windows)
pdsk-util --version    # print version
pdsk-util -h           # show help
```
